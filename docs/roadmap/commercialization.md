# 商业化策略

## 概述

Open-RetroSight 核心软件开源，但围绕它可以构建可持续的商业生态。

## 商业模式：开源核心 + 增值服务

```
                    ┌─────────────────────────────────┐
                    │         云平台订阅（SaaS）        │
                    │   多设备管理 / 数据存储 / 报警    │
                    └─────────────────────────────────┘
                                    │
        ┌───────────────────────────┼───────────────────────────┐
        │                           │                           │
        ▼                           ▼                           ▼
┌───────────────┐         ┌───────────────┐         ┌───────────────┐
│  硬件套件销售   │         │  高级算法插件   │         │   技术支持服务  │
│  开箱即用套装   │         │  复杂仪表模型   │         │   定制开发     │
└───────────────┘         └───────────────┘         └───────────────┘
                                    │
                                    ▼
                    ┌─────────────────────────────────┐
                    │      开源核心软件（免费）          │
                    │   基础OCR / 指针识别 / 数据输出   │
                    └─────────────────────────────────┘
```

## 收入来源一：硬件套件销售

### 产品定位
"开箱即用"的硬件套件，降低用户部署门槛。

### 套件内容

| 组件 | 说明 | 单独购买难度 |
|-----|------|------------|
| 防尘外壳 | 工业级防护 | 需定制 |
| 环形补光灯 | 稳定光源，消除阴影 | 需选型 |
| 万向支架 | 灵活调节角度 | 需适配 |
| 工业摄像头 | 已调试优化 | 需选型 |
| 预装系统 | 开机即用 | 需技术 |

### 价格策略

| 套件 | 内容 | 建议售价 | 毛利率 |
|-----|------|---------|--------|
| 入门套件 | 摄像头+支架+外壳 | ¥299 | 40% |
| 标准套件 | +补光灯+树莓派 | ¥599 | 45% |
| 专业套件 | +工业摄像头+预装系统 | ¥999 | 50% |

### 核心价值
- **工业现场光线复杂，补光灯是刚需**
- 降低用户试错成本
- 提供一致的使用体验

## 收入来源二：高级算法插件

### 基础版（免费）
- 标准七段数码管 OCR
- 基础指针识别
- 单色指示灯识别

### 高级插件（付费）

| 插件 | 功能 | 定价 |
|-----|------|------|
| 复杂数字识别 | 多指针水表、模糊屏幕 | ¥199/设备 |
| 高精度指针 | 非线性刻度、多指针 | ¥299/设备 |
| 多目标识别 | 单摄像头多表盘 | ¥399/设备 |
| 定制模型 | 特定设备专用模型 | 按需报价 |

### 授权模式
- 按设备授权
- 年度订阅
- 永久授权

## 收入来源三：云平台订阅（SaaS）

### 服务内容

| 功能 | 免费版 | 基础版 | 专业版 |
|-----|--------|--------|--------|
| 设备数量 | 1台 | 10台 | 无限 |
| 数据存储 | 7天 | 90天 | 1年 |
| 报警推送 | 邮件 | 邮件+短信 | 全渠道 |
| 数据导出 | - | CSV | API |
| 数据分析 | - | 基础报表 | 高级分析 |
| 技术支持 | 社区 | 工单 | 专属 |

### 定价

| 版本 | 月费 | 年费（8折） |
|-----|------|-----------|
| 基础版 | ¥99/月 | ¥950/年 |
| 专业版 | ¥299/月 | ¥2,870/年 |
| 企业版 | 按需报价 | - |

| 维度 | 传统方案 | Open-RetroSight |
|-----|---------|-----------------|
| 成本 | 数万元 | 数百元 |

| 风险 | 对策 |
|-----|------|
