# 项目介绍

## 背景

中小企业数字化转型面临的"第一公里"难题——老旧设备的数据采集成本过高。传统方案需要：
- 购买昂贵的PLC通讯模块
- 破解封闭协议
- 厂家上门修改底层代码
- 动辄数万元成本
- 停机风险

## 什么是 Open-RetroSight

Open-RetroSight 是一款**非侵入式**的工业边缘AI网关软件，通过计算机视觉技术将传统"哑设备"的数据数字化。

**核心口号**："不拆机、不停产、不改线，给老机器装上'数字眼睛'。"

## 核心价值

| 对比项 | 传统方案 | Open-RetroSight |
|-------|---------|-----------------|
| 成本 | 数万元 | < 300元 |
| 部署时间 | 数天至数周 | 几分钟 |
| 停机风险 | 高 | 零 |
| 技术门槛 | 需专业人员 | 普通人可操作 |

## 适用场景

Open-RetroSight 支持四大类应用场景（由易到难）：

### 1. 七段数码管/LCD屏幕读取
- **应用**：老式注塑机、温控仪、电子秤、数控机床
- **数据**：温度、压力、计数、良品率
- [详细文档](../use-cases/digital-display.md)

### 2. 指针式仪表读取
- **应用**：化工厂、锅炉房的压力表、流量计、电压/电流表
- **技术**：计算机视觉识别指针角度，映射为数值
- [详细文档](../use-cases/analog-gauge.md)

### 3. 状态指示灯识别
- **应用**：产线机台的Andon灯（红/黄/绿）
- **功能**：判断设备状态，计算OEE（设备综合效率）
- [详细文档](../use-cases/indicator-light.md)

### 4. 物理开关/旋钮位置识别
- **应用**：老式配电柜、控制台
- **功能**：识别档位状态（自动/手动模式）
- [详细文档](../use-cases/switch-knob.md)

## 目标用户

### 对企业
投入几百元就能把几十万的旧机床接入互联网，ROI（投资回报率）极高。

### 对开发者
技术栈（Python/CV）成熟，容易上手，但做到高稳定性有挑战，适合练手也适合创业。

## 项目意义

Open-RetroSight 不仅是一个工具，更是连接物理世界（旧工业）与数字世界（新AI）的一座低门槛桥梁。

项目完全符合"推动智能装备迭代"和"加速中小企业数字化改造"的政策方向。
